## 目的
把 `MTC_复杂Excel模型.xlsx` 从“单一示例模型”升级为可复用的 **Excel MTC（More Than Code）武器库**：不仅能算，还能解释、能防呆、能联动外部资产，便于后续继续加模块。

## 已新增/增强的能力模块（2026-04-15）

### 1) Data（明细表）增强
- 将明细区域封装为 **Excel Table（ListObject）**：`tblData`（支持筛选/排序/表格样式）。
- 新增条件格式示例：收入列 `F2:F361` 增加 ColorScale（用于演示“异常/高低值可视化”）。
- 单价公式升级：在原“基准单价×通胀指数”基础上，新增情景折扣项：
  - `*(1-Assumptions!$B$12)`

### 2) Assumptions（假设表）增强：情景（Scenario）驱动
- 新增情景参数表（基准/乐观/压力）：
  - 通胀/提价率（年）
  - 折扣率
  - 备注说明
- 兼容旧模型：将 `B10`（通胀/提价率）改为引用 `Analysis!B4` 的情景驱动公式（不破坏 Data 侧引用）。
- 新增 `B12` 折扣率（同样情景驱动），供 Data 单价公式引用。

### 3) Analysis（仪表盘）增强：控制面板 + 外部联动
- 控制面板新增项：
  - 区域选择 `B3`（下拉）
  - 情景选择 `B4`（下拉）
  - 产品选择 `D3`（下拉）
- 新增本地资源联动入口（超链接）：
  - `interactive/3D_可交互销售仪表盘.html`
  - `assets/MTC_复杂Excel模型.pdf`
  - `MTC_能力演示.pptx`
- KPI 区域新增条件格式（颜色刻度）：同比/波动率“好坏”直观化。
- 新增第三张图表：
  - `月度收入趋势（区域+产品）`（折线图，跟随 区域+产品 选择联动）。

### 4) QualityGate（新增 Sheet）：数据质量门禁 / Poka‑Yoke
- 提供最小可核验的质量门禁：
  - 缺失值、销量<=0、收入<=0 等检查项
  - 每项 PASS/FAIL + 总状态（PASS/FAIL）
- 用于演示“Excel 也能做质量拦截/审计”，并作为后续扩展入口（可继续加重复值、越界、异常分布等）。

### 5) Explain（新增 Sheet）：方法说明（给人看的 Excel）
- 解释模型结构、关键公式链路、以及“情景驱动”的含义。
- 插入装饰图（来自 assets）：`3D_图表旋转封面.png`（用于演示图片嵌入能力）。
