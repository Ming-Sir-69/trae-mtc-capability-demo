## PPT 设计系统（用于本次演示稿）
本设计系统由内置 pptx skill 的 design search 生成（关键词：futuristic data dashboard ai assistant），用于确保 PPT 不是“白底黑字流水账”。

### 推荐风格
- Style：Data-Dense Dashboard（数据密集仪表盘）
- Pattern：Data-Dense + Drill-Down

### 配色（核心）
- Primary：#1E40AF（深蓝）
- Secondary：#3B82F6（亮蓝）
- 强调/CTA：#F59E0B（琥珀）
- Background：#F8FAFC（极浅灰蓝）
- Text：#1E3A8A（深蓝文本）

### 字体（在 PowerPoint 中将按“可用字体”自动降级）
- Header/强调：Fira Sans（或 Calibri/Arial 作为降级）
- 数字/代码风：Fira Code（或 Consolas 作为降级）
