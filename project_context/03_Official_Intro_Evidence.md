## 官方介绍（用于核验“我在说什么”）

以下内容来自 TRAE 官方文档的「TRAE SOLO 概述」页面，用于证明：TRAE SOLO 确实存在 MTC 与 Code 双模式、Web 与 Desktop 双端形态，并定位为 AI 原生工作台，覆盖办公与开发场景。

### 关键摘录（原文）
- “TRAE SOLO 是一款 AI 原生工作台，提供网页版和桌面版，并设有为不同用户群体设计的 More Than Coding (MTC) 与 Code 双模式。”  
- “MTC 模式：面向非开发相关的用户……尤其适合处理文档、数据、演示稿等。”

### 与本次演示的对应关系
- 本次交付的 `MTC_复杂Excel模型.xlsx` 与 `MTC_能力演示.pptx`：对应官方所述 MTC 模式的典型场景（文档/数据/演示稿）
- 本次交付的 `interactive/*.html`：对应“可定制的 AI 协作体验”中的一种可核验扩展形态（本地交互式图表）
